ברוך הבא לפלטפורמת HF

המדריך הזה מכסה את כל הפיצ'רים הזמינים עבורך כמאמן — מניהול לקוחות ותוכניות, דרך מעקב התקדמות וצ'אט, ועד למיתוג האישי של האזור שלך. כל מה שצריך להפעיל כדי לתת ללקוחות שלך חוויה מקצועית ותוצאות אמיתיות.

👥
ניהול לקוחות
הוסף, ערוך ועקוב אחרי כל לקוח — פרטים, מסלול, תאריכים וסטטוס.
🏋️
תוכנית אימונים
בנה תוכניות מותאמות אישית עם AI, ימים, תרגילים וסטים.
🥗
תפריט תזונה
ארוחות, יעדי מאקרו, ואפשרויות חלופה — כולל AI תזונה.
🔄
ניהול סייקל
הגדר סטטוס (טבעי/TRT/בלאסט וכו'), מטרה ופרוטוקול לכל לקוח.
📈
מעקב התקדמות
גרפים של משקל, % שומן ומסה רזה. חישוב בסט קונדישן אוטומטי.
💬
צ'אט וצ'ק-אין חדש
צ'אט 1:1 בזמן אמת + שאלון צ'ק-אין שבועי מהלקוח.
🎨
מיתוג אישי
לוגו, תמונת רקע, צבע מותג וסלוגן — יוחלו על דשבורד הלקוח.
🔔
Push Notifications
שלח התראות מיידיות ללקוחות — ידניות או מתוזמנות.
כניסה ראשונה

🚀 התחלה מהירה

אחרי שהאדמין הפעיל את חשבונך, כך תתחיל:

1
כנס לפלטפורמה
עבור אל login.html ← הכנס אימייל + סיסמה ← תועבר לפאנל המאמן
2
הגדר מיתוג אישי חדש
לחץ 🎨 מיתוג בפאנל ← העלה לוגו + תמונת רקע ← בחר צבע מותג + סלוגן ← שמור
3
הגדר מסלולי תמחור
לחץ 📋 מסלולים ← הזן מחירים ותיאורים ← שמור. הלקוחות יראו את המסלולים בדף חידוש מנוי.
4
הוסף לקוח ראשון
לחץ + מתאמן בסרגל הצד ← מלא שם, אימייל, תאריכי ליווי ← לחץ הוסף
5
בנה תוכנית + תפריט
בחר לקוח ← השתמש ב-AI לבניית תוכנית ← חזור על אותו הדבר לתזונה ← שמור
ניהול

👥 לקוחות

הוספת לקוח חדש

1
לחץ + מתאמן
בסרגל הצד הימני למעלה
2
מלא את הטופס
שם מלא, אימייל (זהו ה-ID של הלקוח במערכת), תחילת וסיום ליווי, סוג מסלול
3
הגדר סיסמה ללקוח
בפאנל הניהול של Supabase Auth — צור משתמש עם האימייל שהזנת ← הלקוח יכנס ב-login.html עם הסיסמה

עריכת פרטים

בחר לקוח מהסרגל הצדדי ← בסעיף "פרטי מתאמן" ניתן לעדכן שם, טלפון, תאריכי ליווי, יעדי מאקרו, יעד שתייה ומשקל עדכני.

כרטיס קשר

לחץ על 👤 כרטיס קשר לראות מייל, טלפון, תאריכי ליווי ומסלול — עם כפתורי WhatsApp ומייל מהיר.

💡 טיפ: ניתן לחלוק לקוחות בין מאמנים. אדמין יכול לשנות את שדה coach_email בלקוח כדי להעביר אותו.
תכנון

🏋️ תוכנית אימונים

בניה ידנית

לחץ + הוסף יום ← תן שם ליום ← בחר קבוצות שרירים ← הוסף תרגילים עם סטים + חזרות + הנחיה.

✨ בניה עם AI

⚡ מהיר
1
הגדר תבנית ברירת מחדל
כתוב את מבנה ה-Split המועדף עליך (דוגמה: "4 ימים PUSH/PULL/LEGS/UPPER") — ישמש כבסיס ל-AI
2
כתוב בקשה
תאר מטרה, ציוד, ימי אימון, רמה, פציעות. ניתן לצרף תמונה של תוכנית קיימת.
3
לחץ ✨ צור תוכנית
AI יבנה תוכנית מלאה תוך שניות ← ערוך לפי הצורך ← שמור

העתקה בין לקוחות

לחץ 📋 העתק תוכנית ← בחר לקוח אחר ← לחץ 📋 הדבק תוכנית. ניתן להעתיק גם יום בודד.

💡 חוסך זמן: תבנית ברירת המחדל שמירה לא ספציפית ללקוח — כשה-AI מקבל בקשה כללית הוא ישתמש בה. בקשה ספציפית תדרוס את התבנית.
תכנון

🥗 תפריט תזונה

בנה ארוחות עם מוצרי מזון, כמויות ומאקרו. הגדר יעדי קלוריות וחלבון/פחמימה/שומן בפרטי הלקוח.

✨ AI תזונה

כתוב: "לקוח של 80 ק"ג, מטרת חיטוב, 2200 קלוריות, ישראלי, אוכל כשר" ← AI יבנה תפריט יומי מלא.

חלופות

כל מוצר מזון יכול לכלול עד 4 חלופות — הלקוח יכול להחליף בזמן אכילה. מגדיל הצלחה תזונתית משמעותית.

✓ מומלץ: הגדר יעד מים יומי (ל׳) בפרטי הלקוח — יופיע בדשבורד עם מד שתייה אינטראקטיבי.
פיצ'ר 1

🔄 ניהול סייקל

הגדר את הסטטוס הנוכחי, המטרה, תאריך התחלה ופרוטוקול פרטי לכל לקוח. הלקוח רואה רק סטטוס ומטרה — הפרוטוקול גלוי למאמן בלבד.

סיום סייקל

⚠️ פרטיות: שדה "פרוטוקול / פרטים" מיועד לך בלבד — הלקוח לא רואה אותו בדשבורד.

רשום כל שבוע/שבועיים משקל ו-% שומן ← המערכת מחשבת מסה רזה ומציגה גרף עם 3 קווים.

בסט קונדישן

המערכת מחשבת אוטומטית את הנקודה שבה המסה הרזה הייתה הגבוהה ביותר — זהו "הבסט קונדישן" של הלקוח.

הוספת % שומן לרשומה קיימת

בטבלת ההיסטוריה לחץ + %שומן לצד רשומה שחסר בה ערך שומן ← הזן את הערך ← המסה הרזה תחושב מחדש.

💡 שיטת עבודה: הלקוח מעדכן משקל בדשבורד שלו ← אתה מוסיף % שומן פעם בשבועיים ← המערכת מציגה ללקוח גרף מלא.
פיצ'ר 3 · חדש

💬 צ'אט וצ'ק-אין שבועי

✓ חדש בגרסה 4! ערוץ תקשורת ישיר בין מאמן ללקוח — ללא WhatsApp, ללא מיילים חיצוניים.

צ'אט 1:1 בזמן אמת

👤פעולהמה קורה
🏋️מאמן כותב הודעהמופיעה מיד בדשבורד הלקוח
👤לקוח כותב הודעהמופיעה מיד בפאנל המאמן
👤לקוח מעלה קובץ/תמונהנשמר ב-Storage ונגיש לשניהם
⚡Realtimeעדכון ללא רענון עמוד

צ'ק-אין שבועי

מדי שבוע (מיום שני) הלקוח מקבל שאלון שבועי לדשבורד שלו:

  • ⚡ אנרגיה — סקאלה 1-10
  • 😴 שינה — סקאלה 1-10
  • 🔥 מוטיבציה — סקאלה 1-10
  • ✅ עמידה בתכנית — כמה טוב הלקוח עמד בתזונה + אימונים
  • 💬 הערות — שאלות, קשיים, הצלחות

אחרי שליחה — הלקוח רואה סיכום, ואתה רואה היסטוריה של עד 8 שבועות אחרונים עם ממוצעים.

💡 שימוש יעיל: בדוק את הצ'ק-אינים בתחילת השבוע ← התאם תוכניות לפי המשוב ← שלח הודעה בצ'אט לתמיכה. זה מגדיל שימור לקוחות משמעותית.

שיחה אוטומטית מצ'ק-אין

כשלקוח שולח צ'ק-אין — נשלחת הודעה אוטומטית לצ'אט עם סיכום הציונים. כך אתה רואה הכל במקום אחד ולא צריך לעבור בין מסכים.

עסקי

💰 מסלולי תמחור ולידים

הגדר עד 3 מסלולי מחיר (חודשי / 3 חודשים / חצי שנה). הלקוחות שסיימו ליווי רואים את המסלולים שלך בדף חידוש מנוי.

ניהול לידים

כשלקוח לוחץ "בחר מסלול" — נוצר ליד חדש ב"לידים ועסקאות". לאחר תשלום — לחץ ✓ שולם לסגירת העסקה.

מעקב הכנסות

לחץ + הוסף הכנסה לתיעוד תשלומים ידניים (מזומן, העברה). ניתן לסנן לפי תאריך ולייצא.

💡 סמן מסלול כ"מומלץ" (checkbox בכרטיס המסלול) — יופיע עם עיצוב מודגש אצל הלקוח.

עזרה

❓ שאלות נפוצות

שאלה

לקוח לא רואה את הלוגו שלי בדשבורד

וודא שסיימת להעלות ושמרת את המיתוג. הלקוח צריך לרענן את הדשבורד. בדוק שה-bucket coach-assets קיים ב-Supabase Storage.

שאלה

הלקוח לא מקבל Push Notifications

הלקוח צריך לאשר הרשאות בפעם הראשונה. נסה לשלוח הודעת test — אם לא מגיע, בקש מהלקוח לסגור ולפתוח מחדש את הדשבורד ולאשר שוב.

שאלה

הצ'אט לא מתעדכן בזמן אמת

ודא שה-Realtime פעיל בטבלת messages ב-Supabase. עבור ל-Table Editor ← messages ← לחץ Enable Realtime.

שאלה

לא מצליח להעלות קובץ גדול בצ'אט

מגבלת העלאה: 10MB לקובץ. קבצי PDF גדולים — דחוס קודם. בעתיד ניתן להגדיל את המגבלה.

שאלה

הלקוח לא רואה את קטע הצ'ק-אין

הצ'ק-אין מופיע רק ללקוחות עם coach_email מוגדר. וודא שהלקוח שויך אליך ושהדשבורד נטען מחדש.

שאלה

רוצה להוסיף מאמן נוסף לפלטפורמה

פנה לאדמין הראשי (הלל) ← הוא יוסיף אותך לטבלת coaches ב-Supabase ← יצור משתמש Auth ← יגדיר לך סיסמה.
